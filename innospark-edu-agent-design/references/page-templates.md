## IAIE Page Templates

### 1. 页面类型选择

先判断属于哪一类：

- 官网首页
- 内容详情页
- 管理后台
- 数据工作台
- Agent 广场
- 单 Agent Demo
- 多代理工作流
- 多模态教学 Studio
- 教学评测与数据报告

### 2. 官网首页

推荐结构：

1. 顶部导航
2. Hero 首屏
3. 研究院价值与定位
4. 核心方向与平台能力
5. 成果与数据指标
6. 新闻动态与活动
7. 合作机构
8. CTA 与页脚

适用规则：

- Hero 可用蓝紫渐变与轨道图形
- 强视觉区不超过 2 段
- 其余段落回到浅色背景和大留白

### 3. 内容详情页

- 结构：栏目标签、标题、元信息、摘要、正文、引用、相关推荐
- 正文宽度：`760px ~ 820px`
- 适合新闻、文章、成果详情、活动详情

### 4. 管理后台

- 结构：顶部导航 / 左侧导航 / PageHeader / 筛选区 / 主内容区
- 默认 `default` 密度
- 如果是列表和配置密集页，可切 `compact`

### 5. 数据工作台

- 结构：筛选器、指标卡、图表区、明细区、说明区
- 页面顶部必须标口径、范围、更新时间
- 图表区按“总览 - 细分 - 证据”排列

### 6. Agent 广场

用于平台首页或 demo 列表页。

推荐结构：

1. 标题与平台价值
2. 搜索
3. 场景筛选
4. Agent 卡片网格
5. 推荐能力区
6. 最近体验

每张 Agent 卡片至少有：

- 名称
- 一句话价值
- 适用对象
- 输入模态
- 输出模态
- 试用入口

### 7. 单 Agent Demo

默认使用 3 区布局：

- 左：输入与配置
- 中：主输出
- 右：来源、说明、导出、分享

必须包含：

- 场景标题与适用对象
- 输入方式与资源来源
- 核心输出
- 证据与依据
- 可继续操作
- 风险提示或边界说明

### 8. 多代理工作流

推荐结构：

1. 任务简介
2. 节点流程图区
3. 当前步骤详情
4. 中间产物列表
5. 最终输出

节点必须区分：

- `待执行`
- `运行中`
- `已完成`
- `异常`
- `已跳过`

### 9. 多模态教学 Studio

推荐结构：

1. 资源输入区
2. 主预览区
3. 时间轴或片段导航
4. 摘要与分析区
5. 证据定位区
6. 教学建议区

### 10. 教学评测与数据报告

推荐结构：

1. 报告标题与基本信息
2. 核心结论摘要
3. 指标概览
4. 维度图表
5. 证据片段或样本
6. 改进建议与导出

### 11. 模板选用规则

- 面向第一次体验：优先 `Agent 广场` 或 `单 Agent Demo`
- 面向流程协作：优先 `多代理工作流`
- 面向教学分析：优先 `多模态教学 Studio` 或 `教学评测与数据报告`
- 面向管理：优先 `管理后台` 或 `数据工作台`
