# Multimodal Teaching Studio Template

## 页面目标

- 统一接收多模态教学资源
- 提供主预览、摘要、证据与建议
- 支持从结论反查到原始资源

## 页面结构

1. 资源输入区
2. 资源预览区
3. 时间轴或片段导航
4. AI 摘要与分析区
5. 证据定位区
6. 教学建议区

## 推荐组件

- MultimodalAssetCard
- Timeline
- InsightCard
- SourcePanel
- ActionFooter

## 页面检查

- 文档、图片、音频、视频是否都能统一预览
- 是否能从分析结果跳回页码或时间点
- 是否为教师保留足够清晰的操作路径
