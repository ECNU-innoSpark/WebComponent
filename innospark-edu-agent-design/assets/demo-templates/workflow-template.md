# Multi-Agent Workflow Template

## 页面目标

- 展示多代理协作流程
- 让用户理解每个节点做了什么
- 让用户看到中间产物与最终输出

## 页面结构

1. 任务简介
2. 节点流程图区
3. 当前步骤详情
4. 中间产物列表
5. 最终输出与导出

## 推荐组件

- WorkflowBoard
- StepFlow
- IntermediateResultList
- ResultComparison
- ActionFooter

## 节点状态

- 待执行
- 运行中
- 已完成
- 异常
- 已跳过

## 页面检查

- 节点依赖关系是否清楚
- 输入 / 动作 / 输出是否一一对应
- 结果是否可追溯到具体节点
